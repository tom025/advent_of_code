# AoC 2025 #

My solutions to the Advent of Code 2025

## Development ##

Install dependencies and create venv

```shell
uv sync
```

Activate environment on Linux and macOS

```shell
source .venv/bin/activate
```

Run a solution for day

```shell
python src/2025/day_01.py
```

Run tests
```shell
pytest
```

Run tests for day
```shell
pytest test/test_day01.py
```
